## 本周 UI 保真验收清单（开发前置，ones，硬性必达）

- Space 主路径为三栏：`sidebar`（左） / `content-area`（中，含 `content-toolbar`） / `rightbar`（右）；右栏宽度与收起行为与原型层级一致。
- 右栏与正文之间使用 `rightbar-splitter` 单柄切换收起/展开（icon-only），hover 高亮可辨；收起态 `rightbar` 具备稳定 class/状态（如 `is-collapsed`）且正文区宽度随之变化。
- 右栏含 `rightbar-top-divider`（或等价 DOM）使顶部分割线与内容区顶栏分割对齐；`rightbar-section` 内「页面大纲」含 `toc` 容器；「页面信息」使用 `page-info-list`（或等价结构）展示标签与值。
- 「页面信息」字段：创建人、创建时间、最新修改时间；时间格式 `yy-mm-dd HH:mm:ss`；不展示 Space ID 等与用户无关的调试字段。
- 左侧树使用 `tree-node` / `tree-children` 结构；顶栏齿轮菜单使用 `header-menu-trigger` / `header-menu` 等可验收结构。
