## 本周 UI 保真验收清单（开发前置，ones，硬性必达）

> 本周 UI 聚焦登录页（参考稿风格），不涉及 Space 三栏原型对齐项。

- 登录页采用左右分栏：左侧为视觉互动区，右侧为表单卡片；卡片圆角与阴影层级稳定。
- 左侧存在 `logo1.png` 品牌露出：位于左上角，带圆角；大小适中，不遮挡角色主体。
- 右侧标题与副标题存在且文案一致：标题为“元体WIKI，欢迎回来！”，副标题为“请输入你的账号信息”。
- 表单字段存在：账号输入框、密码输入框（含显示/隐藏按钮），记住账号 checkbox，忘记密码入口（仅提示联系管理员）。
- 内网口径：不出现“注册/Sign Up/Google 登录”相关入口与按钮。
- 互动角色：四个角色的眼睛跟随鼠标视线移动；账号/密码聚焦时存在差异化反馈（紫色在聚焦时“探头/偷窥”更明显）。
