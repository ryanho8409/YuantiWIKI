# Release V1.2 说明

> **发布日期**：2026-04-01  
> **相对基线**：Release V1.1（2026-03-25）

本版本聚焦 **登录页体验升级**（对标参考稿的分栏布局 + 互动角色），并补齐深色模式下品牌 Logo 的一致性与自动化回归，形成可发布的新特性版本。

---

## 1. 本日下午开发总结（可贴 GitHub Release Notes）

### 1.1 前端（登录页新体验）

- **分栏登录页**：`/login` 改为左右分栏布局（左：互动角色与品牌露出；右：表单卡片）。  
- **内网口径**：移除 Google 登录入口；不提供注册入口，底部提示“联系管理员开通”。  
- **文案收口**：标题为“元体WIKI，欢迎回来！”，副标题为“请输入你的账号信息”。  
- **表单交互**：账号/密码输入密度更紧凑；密码支持显示/隐藏；保留“30 天内记住账号”与“忘记密码？”提示。  
- **互动角色**：四个几何角色支持视线跟随与聚焦反馈；账号/密码聚焦时紫色角色“偷窥”动作差异化（账号更明显、密码更小幅）。  
- **浏览器自动填充观感**：抑制浏览器 autofill 引入的蓝色底色，保证输入框观感与设计一致。

### 1.2 品牌与主题一致性（深色模式）

- **登录页与系统顶栏 logo**：浅色模式使用 `logo1.png`；深色模式使用 `logo_dark.png`（均位于 `frontend/public/`，并用 `BASE_URL` 拼接避免子路径部署裂图）。

### 1.3 测试与回归

- 新增 E2E：`frontend/tests/e2e/week11.e2e.spec.mjs`  
  - 断言 **dark 模式** 下 `/login` 左侧 logo 与登录后顶栏 logo 均为 `logo_dark.png`。
- 测试计划回填：`tests/week11/week11-test-plan.md` 已补充 `A-05`（自动化）与 `M-06`（手测项）入口与结果。
